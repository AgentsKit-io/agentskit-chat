# Vue support reference

Runs the shared support, onboarding, operations, and RAG definitions through the
native Vue shell (`@agentskit/chat/vue`).

```bash
pnpm --filter @agentskit/chat-example-vue dev
```

See [DOM renderer parity](../../docs/examples/dom-renderer-parity.md) and the
[support reference](../../docs/examples/support-reference.md).
