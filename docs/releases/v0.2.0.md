# AgentsKit Chat 0.2.0

AgentsKit Chat 0.2 adds the public backend vertical needed to run the same
cross-framework chat definition against hosted or self-hosted Ask services.
AgentsKit remains the controller, message, memory, cancellation, and renderer
substrate.

## Included

- A runtime-validated Ask backend contract with session-aware requests,
  deterministic escalation, cited retrieval, persistence, typed diagnostics,
  cancellation, rate-limit composition, and privacy-safe baseline metrics.
- A Web-standard server implementation and deployment guidance for hosted and
  self-hosted environments.
- Shared adapter tolerance for valid Ask NDJSON delivered as `text/plain`,
  while ordinary text retains the existing safe fallback.
- The fixed 12-package graph, seven native renderer implementations, and the
  existing protocol, accessibility, browser, native, PTY, package, and
  provenance release gates.

Existing `0.1.x` chat definitions remain source-compatible. Upgrade the whole
fixed package graph together and follow the
[0.2 migration guide](./migration-to-0.2.md).
